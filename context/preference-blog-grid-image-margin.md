# Blog Grid Image Styling

## Type
Preference

## Date
2026-02-17

## Description
The user prefers the images in the blog grid to have no top margin, making them flush with the top of the article card.

## Context/Reasoning
The user mentioned that it "looks better" without the top margin.

## Actionable Item
Always ensure `.article-entry img` in `BlogGrid` has `margin: Margin.zero` and is correctly positioned at the top of the card.
